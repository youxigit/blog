QQ日迹涂鸦优化

  • IOS

前言

IOS6.6.0版本手机QQ，日迹涂鸦效果采用了矢量图形内存存储 + 局部刷新算法 + 二阶贝塞尔曲线中点绘制算法 + Quartz 2D渲染的全新解决方案，实现曲线平滑、无锯齿的效果。新方案主要优势是：矢量图形内存存储避免了对Disk进行频繁的I/O操作、局部刷新算法减少了CPU计算渲染路径和GPU渲染图形的负荷、Quartz 2D克服了OpenGL晦涩难懂的不足又能轻易的胜任图形渲染工作。阅读本文你可以快速的掌握绘制优雅线条的要点。

本文主要包含以下几个部分：

  • 优化背景
  • 优化前后效果
  • 优化方案详解
  • 总结
  • 参考文献

优化背景

绘制线条从来都不是一件困难的事情，但是要采用高效、轻量级别的方案绘制优雅的线条也不是一件很简单的事情。当时在开发过程中项目非常急迫，分别尝试了几种不是很优的解决方案。

方案一：把矢量路径存储在内存中，在手指移动的过程中记录用户移动的位置同时把当前路径渲染出来刷新屏幕。缺点非常明显，当单次操作持续时间长、移动速度比较快、线条曲率较高的情形下线条不平滑、线条连接处出现明显、生硬的折线。原因是快速移动的时候全局渲染CPU、GPU负荷很重，就出现掉点导致线条不平滑。好处是路径存储在内存当中不需要I/O操作。显然这种效果是不能满足需求的。

方案二：每一次手指操作结束后把当前的图形写入Disk保存，下一次操作的时候计算操作的区域，然后使用局部更新机制避免全局渲染带来的CPU、GPU资源浪费。用户撤消的时候就从Disk中把上一次操作的图片渲染出来。这种方案的优点是采用了局部刷新机制，撤消功能实现简单。缺点是大量的I/O操作对设备的电量消耗、磁盘读写性能和使用寿命、可用空间都是严峻的挑战。绘制出来的线条虽然比较平滑、无锯齿能满足产品需求，但是痛点却非常明显。

当时迫于时间压力，就采用了方案二完成需求，打算后期版本迭代来优化处理。于是在6.6.0就开始了涂鸦的优化工作。

优化前后效果

优化前的效果

方案一效果：不光滑、锯齿

方案二效果：光滑、无锯齿但是大量I/O操作

优化后的效果

优化方案效果：光滑、无锯齿、高性能

优化方案详解

优化方案主要结合了方案一和方案二的优点：矢量路径内存存储、局部刷新算法、Quartz 2D渲染框架，以及在此基础上改进了矢量路径绘制算法，采用二阶贝塞尔曲线中点绘制算法来达到最终光滑、无锯齿、高性能的优化效果。

接下来结合代码详细讲述优化方案中的主要技术点：

  • 矢量路径内存存储

TBEditDrawStep用来记录单次操作的线条的属性和矢量位置信息，_drawSteps是一个数组用来记录所有的线条。

-(void)touchesBegan:(NSSet *)touches withEvent:(UIEvent *)event
{
[self.superview touchesBegan:touches withEvent:event];
    UITouch *touch = [touches anyObject];
/**
 *记录最近的三个点的位置，主要用于中点绘制算法，优化线条连接处的平滑效果
 */
_previousPoint1 = [touch locationInView:self];
_previousPoint2 = [touch locationInView:self];
_currentPoint   = [touch locationInView:self];

    /**
 * TBEditDrawStep 记录用户每一次操作的信息，也可以理解为单次绘制的线条信息
 */
TBEditDrawStep *aStep = [[TBEditDrawStep new] autorelease];
aStep.step = self.drawStep;
aStep.lineWidth = self.lineWidth;
aStep.lineColor = self.lineColor;
aStep.path = [[NSMutableArray new] autorelease];
[_drawSteps addObject:aStep];

[self touchesMoved:touches withEvent:event];
}

-(void)touchesMoved:(NSSet *)touches withEvent:(UIEvent *)event
{
UITouch *touch  = [touches anyObject];

_previousPoint2  = _previousPoint1;
_previousPoint1  = _currentPoint;
_currentPoint    = [touch locationInView:self];

TBEditDrawStep *aStep = [_drawSteps lastObject];
if (aStep.step == TBDrawStepDraw) {
[aStep.path addObject:[NSValue valueWithCGPoint:_currentPoint]];
}
/**
 *计算局部刷新的区域
 */
[self calculateNeedToDrawRect:_previousPoint1 previousPoint:_previousPoint2 currentPoint:_currentPoint];
}
  • 局部刷新算法

首先利用最近三个点计算出贝塞尔曲线的轨迹，然后计算出包含轨迹的最小矩形区域CGRect bounds = CGPathGetBoundingBox(path);，这个区域就是需要重新绘制的区域。执行[self setNeedsDisplayInRect: bounds];就实现了局部刷新的效果。

- (void)calculateNeedToDrawRect:(CGPoint)pp1 previousPoint:(CGPoint)pp2 currentPoint:(CGPoint)cp
{
    /**
     *取中点绘制，优化连接处平滑度。
     */
        CGPoint mid1    = midPoint1(pp1, pp2);
    CGPoint mid2    = midPoint1(cp, pp1);

    //计算局部刷新的区域
    CGMutablePathRef path = CGPathCreateMutable();
    CGPathMoveToPoint(path, NULL, mid1.x, mid1.y);
    CGPathAddQuadCurveToPoint(path, NULL, pp1.x, pp1.y, mid2.x, mid2.y);
    CGRect bounds = CGPathGetBoundingBox(path);
    CGPathRelease(path);

    CGRect drawBox = bounds;

    //考虑线条宽度的因素，得到准确的刷新区域
    drawBox.origin.x        -= self.lineWidth * 1;
    drawBox.origin.y        -= self.lineWidth * 1;
    drawBox.size.width      += self.lineWidth * 2;
    drawBox.size.height     += self.lineWidth * 2;

    //先渲染当前的layer
    [self.layer renderInContext:UIGraphicsGetCurrentContext()];

    //局部刷新
    [self setNeedsDisplayInRect:drawBox];

    //http://stackoverflow.com/a/4766028/489594
    [[NSRunLoop currentRunLoop] runMode:NSDefaultRunLoopMode beforeDate: [NSDate date]];
}
  • 二阶贝塞尔曲线中点绘制算法

虚线表示取中点修正后的效果，对比差异效果非常明显。

    - (void)touchesBegan:(NSSet *)touches withEvent:(UIEvent *)event
{
    ctr = 0;
    UITouch *touch = [touches anyObject];
    pts[0] = [touch locationInView:self];
}

- (void)touchesMoved:(NSSet *)touches withEvent:(UIEvent *)event
{
        UITouch *touch = [touches anyObject];
    CGPoint p = [touch locationInView:self];
    ctr++;
    pts[ctr] = p;
    if (ctr == 4) 
    {
    pts[3] = CGPointMake((pts[2].x + pts[4].x)/2.0, (pts[2].y + pts[4].y)/2.0); // move the endpoint to the middle of the line joining the     second control point of the first Bezier segment and the first control point of the second Bezier segment 

    [path moveToPoint:pts[0]];
    [path addCurveToPoint:pts[3] controlPoint1:pts[1] controlPoint2:pts[2]]; // add a cubic Bezier from pt[0] to pt[3], with control points pt[1] and pt[2]

    [self setNeedsDisplay];
    // replace points and get ready to handle the next segment
    pts[0] = pts[3]; 
    pts[1] = pts[4]; 
    ctr = 1;
    }

}

- (void)touchesEnded:(NSSet *)touches withEvent:(UIEvent *)event
{
    [self drawBitmap];
    [self setNeedsDisplay];
    [path removeAllPoints];
    ctr = 0;
}

- (void)touchesCancelled:(NSSet *)touches withEvent:(UIEvent *)event
{
    [self touchesEnded:touches withEvent:event];
}

- (void)drawBitmap
{
    UIGraphicsBeginImageContextWithOptions(self.bounds.size, YES, 0.0);

    if (!incrementalImage) // first time; paint background white
    {
    UIBezierPath *rectpath = [UIBezierPath bezierPathWithRect:self.bounds];
    [[UIColor whiteColor] setFill];
    [rectpath fill];
    }
    [incrementalImage drawAtPoint:CGPointZero];
    [[UIColor blackColor] setStroke];
    [path stroke];
    incrementalImage = UIGraphicsGetImageFromCurrentImageContext();
    UIGraphicsEndImageContext();
}
  • Quartz 2D渲染

在需要刷新的时候调用[self setNeedsDisplayInRect:drawBox],drawBox表示重绘的区域。

自动执行- (void)drawRect:(CGRect)rect函数进入绘制阶段。

- (void)drawRect:(CGRect)rect
{
    [self paintLine];
}

然后调用Quartz 2D的接口开始重新绘制界面

- (void)paintLine
{
    CGPoint mid1 = midPoint1(_previousPoint1, _previousPoint2);
    CGPoint mid2 = midPoint1(_currentPoint, _previousPoint1);

    CGContextRef context = UIGraphicsGetCurrentContext();

    [self.layer renderInContext:context];

    CGContextMoveToPoint(context, mid1.x, mid1.y);
    CGContextAddQuadCurveToPoint(context, _previousPoint1.x, _previousPoint1.y, mid2.x, mid2.y);
    CGContextSetLineCap(context, kCGLineCapRound);
    CGContextSetBlendMode(context, kCGBlendModeNormal);
    CGContextSetLineJoin(context, kCGLineJoinRound);
    CGContextSetLineWidth(context, self.lineWidth);
    CGContextSetStrokeColorWithColor(context, self.lineColor.CGColor);
    CGContextSetShouldAntialias(context, YES);
    CGContextSetAllowsAntialiasing(context, YES);
    CGContextSetFlatness(context, 0.1f);

    CGContextSetAlpha(context, self.lineAlpha);
    CGContextStrokePath(context);
}

以上是Quartz 2D渲染工作的完整流程。
